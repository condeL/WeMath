2.5 Rules of Differentiation 

In order to avoid differentiating functions from first principles, we can establish certain rules. Rule 1 If f is a constant function, f(x) = c, then f 0 (x) = 0 67 Rule 1 may also be written as

d/(dx ) c = 0

This result is geometrically evident if one considers the graph of a constant function. This is an horizontal line, which has slope 0.

Rule 2: The Power Rule 

If f(x) = xn, where n is an integer, then

f 0 (x) = nx(n-1)

The Power Rule may also be written as

d/(dx ) (xn) = nx(n-1)
